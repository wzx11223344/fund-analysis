# fund-analysis -- 中国公募基金智能分析工具

## 能力边界

**支持的操作：**

- 关键词搜索公募基金（股票型、混合型、债券型、货币型等全部类型）
- 获取基金基本信息（基金经理、规模、费率、成立时间、评级等）
- 单只基金全面分析：年化收益、年化波动率、最大回撤、Sharpe/Sortino/Calmar 比率、VaR/CVaR、Alpha/Beta、跟踪误差、胜率、盈亏比
- 两只基金横向对比
- 三只及以上基金的 Markowitz 均值-方差组合优化（有效前沿、最小方差组合、最大 Sharpe 切线组合、风险平价、Black-Litterman 模型、蒙特卡洛模拟）
- 生成 Markdown 格式完整分析报告
- 导出基金历史净值 CSV
- 本地缓存管理，避免重复请求

**不支持的操作：**

- 实时交易下单或自动调仓
- 盘中净值估算（仅有日频历史净值）
- 私募基金或海外基金数据
- 技术面 K 线分析或图表生成
- 宏观经济环境判断或择时建议
- 任何投资建议（仅提供数据与分析结果）

## 触发条件

当用户提出以下意图时触发本 Skill：

- "帮我分析这只基金 000001"
- "搜索关键词'新能源'的公募基金"
- "比较基金 A 和基金 B 哪个好"
- "帮我做基金组合优化"
- "生成基金分析报告"
- "导出基金净值数据"
- 任何涉及中国公募基金的搜索、分析、比较、组合优化需求

## 使用方法

```bash
# 搜索基金
python fund_cli.py search <关键词>

# 查看基金基本信息
python fund_cli.py info <基金代码>

# 单只基金全面分析
python fund_cli.py analyze <基金代码>

# 两只基金对比
python fund_cli.py compare <代码1> <代码2>

# 三只基金组合优化
python fund_cli.py optimize <代码1> <代码2> <代码3>

# 生成 Markdown 报告
python fund_cli.py report <基金代码>

# 导出净值 CSV
python fund_cli.py export <基金代码>

# 清空缓存
python fund_cli.py clearcache
```

## FAQ

**Q1: 数据来源是什么？需要 API Key 吗？**
数据通过 akshare 开源库获取，完全免费，无需注册任何 API Key。数据源为东方财富、天天基金等公开平台。

**Q2: 支持哪些基金类型？**
覆盖全部开放式公募基金类型：股票型、混合型（偏股/偏债/平衡）、债券型、货币型、指数型、ETF、QDII、FOF 等。

**Q3: 组合优化需要多少只基金？**
最少 3 只，理论上限取决于历史净值数据的起始日期一致性。通常 3-10 只效果最佳。

**Q4: 分析报告包含哪些内容？**
基本信息、收益指标、风险指标、风险调整后收益、回撤分析、滚动统计、基金经理信息、持仓明细（如有），全部输出为 Markdown 格式。

**Q5: 为什么某些基金搜索不到？**
可能原因：该基金已清盘或合并、为场内 ETF（需用场内代码）、或 akshare 接口返回的基金列表不包括该类基金。可尝试用基金名称全称搜索。

## 技术栈

- **语言**: Python 3.9+
- **数据源**: akshare（东方财富/天天基金数据）
- **科学计算**: NumPy, Pandas, SciPy（最优化）
- **核心模块**: `fund_analyzer/` (fetcher, analyzer, portfolio, reporter)
- **CLI 入口**: `fund_cli.py`
